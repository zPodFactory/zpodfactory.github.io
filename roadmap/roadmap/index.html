    Roadmap

    READ THIS

    Please keep in mind we work on this project whenever we have free time, so there will be no ETA/Dates provided for any of the items below.

    When it's done™

    Planned features and Current Status

    Priority Name Description Status
    Library & Components Enhance greatly the components management, lifecycle, downloads, misc features MVP Done
    API / SDK zPodFactory should be an API endpoint so we can build a CLI, a Slackbot, etc on top of it MVP Done
    Flow Engine Enhance the scheduling/orchestration capabilities, and especially improve UX for troubleshooting with visualization MVP Done
    Profiles Enhance zPod 1.0 profile model, making it more reliable and extensible MVP Done
    zPod Deploy & manage a zPod lifecycle MVP Done
    Endpoints Deploy WHERE you want, ANY vCenter/NSX target MVP Done
    CLI Primary way to control everything, using a fancy CLI MVP Done
    Requires improvements
    Documentation Documentation is key, and we need to improve it Requires improvements
    RBAC Support for Users, Groups & Permissions on zPods & Endpoints MVP Done
    Enhanced networking capabilities for complex envs Offer advanced network features such as:
    - Create tagged guest VLANs
    - Manage NSX T1 static routes
    - implement zBox & VyOS network appliances    		 MVP Done

    Work In Progress
    Enhanced storage capabilities for complex envs Offer advanced storage features such as:
    - Create NFS shares
    - Create vSAN clusters    		 Planned
    Slack Bot Ability to control zPodFactory via Slack Cancelled
    Settings Ability to set zPodFactory wide configuration settings MVP Done
    Requires improvements
    Appliance Ability to deploy the zPodFactory framework via an OVA Appliance for simple deployment/setup and clean UX
    - Packer Appliance & scripts
    - OVF Properties
    - Wireguard VPN    		 MVP Done
    User-driven script extensibility Extensibility on zpod and zpod_component objects with hook points:
    - pre-script
    - post-script    		 Planned
    Fancy Feature A Wireguard/Tailscale VPN per zPod MVP Done
    Fancy Feature B Public expose with Let's Encrypt certificates Planned
    Fancy Feature C Dynamic DNS for zPods (VyOS/powerdns combo) MVP Done

    This is not an exhaustive list, and is probably going to change over time quite a lot.
